## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
```

This will create an optimized production build in the `out` directory.

## Deployment

This site is configured for easy deployment to Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy with one click

Alternatively, use the Vercel CLI:
```bash
vercel
```

## Customization

### Personal Information

Update the personal details in `app/page.tsx`:
- Name and title
- Bio and description
- Skills and expertise
- Social media links
- Blog posts

### Styling

Modify colors and themes in `tailwind.config.js` and `app/globals.css`.

### Content

Add more sections or pages by creating new components in the `app` directory.
